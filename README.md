# VoLamTruyenKy

## **Hotkeys:**
 - F1: Cửa sổ hỗ trợ
 - F2: Cửa sổ tùy chọn
 - F3: Cửa sổ Trang bị và Thuộc tính
 - F4: Cửa sổ Hành trang
 - F5: Cửa sổ Kỹ năng võ công
 - F6: Cửa sổ bằng hữu
 - F7: Hiện tên nhân vật
 - F8: Hiện thanh máu nhân vật
 - F9: Bật/tắt PK
 - M: Lên/xuống ngựa
 - T/O: Bật chế độ cho phép giao dịch
 - P: Bật tổ đội
 - V: Ngồi thiền
 - R: Đi bộ/ chạy
 - Phím Tab: Bật/tắt/mở rộng mini map
